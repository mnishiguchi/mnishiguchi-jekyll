Masatoshi Nishiguchi's blog powered by Jekyll etc.

## Dependencies

- ruby
- jekyll
- nodejs
- yarn

## Get started

Make sure dependencies are installed:

```
ruby --version
jekyll --version
node --version
yarn --version
```

For development, run:

```
yarn develop
```

and visit http://localhost:4000

For production:

```
yarn build
```
